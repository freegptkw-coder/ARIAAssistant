# ARIA Assistant Android App

Complete Android application for voice-based AI assistant with root command execution.

## Features
✅ Voice Recognition (Speech-to-Text)
✅ Text-to-Speech (TTS) responses
✅ Chat UI with conversation history
✅ Letta Cloud API integration
✅ Root command execution via JSON parsing

## Building the APK

### Option 1: Using Android Studio (Recommended)
1. Transfer the `ARIAApp` folder to your computer
2. Open Android Studio
3. Click "Open an Existing Project" and select the ARIAApp folder
4. Wait for Gradle sync to complete
5. Click Build → Build Bundle(s) / APK(s) → Build APK(s)
6. APK will be in: `app/build/outputs/apk/debug/app-debug.apk`

### Option 2: Using Gradle Command Line
```bash
cd ARIAApp
./gradlew assembleDebug
```

### Option 3: Build on Android Device (Advanced)
Install Termux and Android build tools:
```bash
pkg install openjdk-17 gradle
export ANDROID_HOME=/path/to/android/sdk
cd /sdcard/123/ARIAApp
gradle assembleDebug
```

## Configuration

### Update Letta API Credentials
Edit `app/src/main/java/com/aria/assistant/LettaApiService.kt`:
```kotlin
private val baseUrl = "https://api.letta.com"
private val agentId = "your-agent-id-here"
```

## Permissions Required
- INTERNET: API communication
- RECORD_AUDIO: Voice input
- ACCESS_SUPERUSER: Root command execution

## Installation
1. Enable "Install from Unknown Sources" in Android settings
2. Transfer APK to device
3. Install APK
4. Grant all permissions including root access

## Usage
1. Launch ARIA Assistant app
2. Tap 🎤 button to speak or type in text box
3. App will:
   - Send your message to Letta API
   - Speak the response using TTS
   - Execute any root commands in JSON format
   - Display conversation history

## Root Command Format
ARIA will parse JSON blocks in responses:
```json
{
  "action": "root_shell",
  "command": "your_shell_command_here"
}
```

## Troubleshooting
- **No voice recognition**: Grant RECORD_AUDIO permission
- **Root commands fail**: Ensure device is rooted and grant superuser permission
- **API errors**: Check internet connection and agent ID configuration
- **Build errors**: Ensure Android SDK is properly installed
